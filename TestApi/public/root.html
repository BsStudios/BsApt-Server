Welcome To BsRepo v1.0.0



BsRepo Is a repo server software for the Bs Software Suite RepoManager.



-----Example API Get Structure-----

Get Supported Architectures:

GET /aritechture HTTP/1.1
Host: *dns_name/ip*
Accept: application/json,text/plain,text/html

Get Supported OS's:

GET /*architecture*/oss HTTP/1.1
Host: *dns_name/ip*
Accept: application/json,text/plain,text/html

Get Release Lists:

GET /*architecture*/*os(windows,macintosh,linux)*/releases HTTP/1.1
Host: *dns_name/ip*
Accept: application/json,text/plain,text/html

Get Release Listing:

GET /*architecture*/*os(windows,macintosh,linux)*/*release_list* HTTP/1.1
Host: *dns_name/ip*
Accept: application/json,text/plain,text/html

Get Latest Release:

GET /*architecture*/*os(windows,macintosh,linux)*/*release_list*/*release_name* HTTP/1.1
Host: *dns_name/ip*
Accept: application/gzip,application/zip,application/x-7z-compressed,application/x-rar-compressed,application/octet-stream

Get Release Versions:

GET /*architecture*/*os(windows,macintosh,linux)*/*release_list*/*release_name*/versions HTTP/1.1
Host: *dns_name/ip*
Accept: application/json,text/plain,text/html

Get Specific Release:

GET /*architecture*/*os(windows,macintosh,linux)*/*release_list*/*release_name*?v=*version_number* HTTP/1.1
Host: *dns_name/ip*
Accept: application/gzip,application/zip,application/x-7z-compressed,application/x-rar-compressed,application/octet-stream



-----Example API Post Structure-----

Add Release List:

POST /*architecture*/*os(windows,macintosh,linux)*/releases HTTP/1.1
Host: *dns_name/ip*
Content-Type: application/x-www-form-urlencoded
Authorization: Basic *hased_urername_and_password*
Content-Length: 11**length_of_content**

name=*name*&operation=*(remove_or_add)*
